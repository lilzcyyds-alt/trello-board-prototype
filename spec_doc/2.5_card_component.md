# 2.5 Card Component

## Summary
A Trello-like card item shown inside a list.

## Requirements
- Displays card title (single or multi-line)
- Visual states:
  - Default
  - Hover (shadow/border)
  - Active/pressed
  - Focus-visible (keyboard)
- ~~Click opens the Card Detail Modal (see 2.6)~~ *(Not implemented - Modal feature was skipped)*
- **Actual behavior**:
  - Hover reveals completion circle with indent animation
  - Click on completion circle triggers pop and blast animations, then removes the card
  - Card supports drag-and-drop functionality

## Acceptance Criteria
- Card matches reference look closely (padding, border radius, shadow)
- Hover/active/focus states are implemented and consistent
- Card can be dragged (see 2.7)
