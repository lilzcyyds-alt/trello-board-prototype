# 2.5 Card Component

## Summary
A Trello-like card item shown inside a list.

## Requirements
- Displays card title (single or multi-line)
- Visual states:
  - Default
  - Hover (shadow/border)
  - Active/pressed
  - Focus-visible (keyboard)
- Click opens the Card Detail Modal (see 2.6)

## Acceptance Criteria
- Card matches reference look closely (padding, border radius, shadow)
- Hover/active/focus states are implemented and consistent
- Card can be dragged (see 2.7)
