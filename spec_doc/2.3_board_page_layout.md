# 2.3 Board Page (Layout + Header)

## Summary
Main board view with header and list columns, positioned to the right of the Inbox column.

## Layout
- Position: Occupies the remaining width to the right of the Inbox panel when both are visible, or full screen within the layout padding.
- Container: Floating rounded panel (`rounded-2xl`) with #424957 border and large shadow.
- Background: Gradient from #6225b8 (top-left) to #b02c67 (bottom-right).
- Scroll: Horizontal scroll for list columns within the panel.

## Requirements
- Header elements:
  - Editable Board title (auto-expanding input based on content)
  - Action buttons (aligned right): Share (with UserPlus icon), More (MoreHorizontal).
  - Background: Black/10 with backdrop-blur.
- Body:
  - Lists/columns rendered horizontally
  - Board background (gradient) and padding similar to Trello
  - Board background and padding similar to Trello

## Acceptance Criteria
- Board page loads with mock board state
- Columns display in correct order (from `listOrder`)
- Horizontal scroll works on smaller screens
