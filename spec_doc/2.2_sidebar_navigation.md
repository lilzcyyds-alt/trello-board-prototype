# 2.2 Sidebar Navigation

## Summary
A persistent left sidebar to switch between Inbox and Board.

## Requirements
- Visible on both Inbox and Board pages
- Contains:
  - App/Workspace label (simple)
  - Nav items: Inbox, Board (or Boards)
- Interaction states:
  - Hover
  - Active/selected
  - Focus-visible for keyboard navigation

## Acceptance Criteria
- Navigation works between `/inbox` and `/board/[id]`
- Selected item reflects current route
- Styling is consistent and matches the reference vibe
