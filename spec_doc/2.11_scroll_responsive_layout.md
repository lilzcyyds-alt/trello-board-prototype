# 2.11 Horizontal Scroll + Layout Responsiveness

## Summary
Ensure the Board UI behaves well across window sizes with horizontal scrolling for lists.

## Requirements
- Lists container:
  - Horizontal scroll when lists overflow
  - Smooth scrolling; avoid layout jumps
- Basic responsiveness:
  - Sidebar collapses (optional) or stays fixed
  - Board header remains readable

## Acceptance Criteria
- On small viewport, lists can be scrolled horizontally
- No overlapping or broken layout during drag actions
