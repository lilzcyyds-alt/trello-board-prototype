# 2.12 UI States & Visual Fidelity Checklist

## Summary
A checklist to ensure pixel-level attention to detail.

## Checklist
- Typography:
  - Base font size and weight
  - Line heights
- Spacing:
  - Card padding
  - Column gap
  - Header padding
- Colors:
  - Backgrounds
  - Borders
  - Hover highlights
- Shadows & borders:
  - Card hover shadow
  - Modal shadow
  - Focus ring
- States:
  - Hover
  - Active/pressed
  - Focus-visible
  - Disabled (if any)
- Motion:
  - Subtle transitions on hover (fast)

## Acceptance Criteria
- Key UI components look and feel coherent and Trello-like
- No "unstyled" elements
