# 2.12 UI States & Visual Fidelity Checklist

## Summary
A checklist to ensure pixel-level attention to detail.

## Checklist
- Typography:
  - Base font size and weight
  - Line heights
- Spacing:
  - Overall layout padding: `p-3`
  - Panel gap: `gap-3`
  - Card padding
  - Column gap
  - Header padding
- Colors:
  - Global Background: #23272e
  - Inbox Panel: #0d3375
  - Inbox Composer Container: #101204
  - Inbox Composer Textarea: #22272b
  - Board Panel: Gradient (#6225b8 to #b02c67)
  - Card Background (All): #252930
  - Card Text: #ffffff
  - Card Source/Hint Text: #828487
  - Primary Blue (Action Buttons): #579dff
  - Hover highlights (Card): #8db1f0 wrapper
  - Hover highlights (General)
- Shadows & borders:
  - Panel Border: #424957
  - Resize Divider Handle: #424957 (Idle), #5a6270 (Hover), #7c8594 (Dragging)
  - Panel Corners: `rounded-2xl`
  - Composer Focus: `ring-[#85b8ff]`
  - Panel Shadow: `shadow-2xl`
  - Card hover shadow
  - Modal shadow
  - Focus ring
- States:
  - Hover (Card): Blue border highlight (#8db1f0), completion circle appears, title indents.
  - Active/pressed
  - Focus-visible
  - Disabled (if any)
- Motion:
  - Subtle transitions on hover (fast).
  - Inbox Item Hover: Title indents (pl-0 to pl-8) and completion circle fades in.
  - Completion Blast: 8-particle burst effect on click (`animate-blast`).
  - Completion Pop: Scale-up and fill animation for circle (`animate-pop`).
  - Item Exit: Sequenced fade and slide-left (`-translate-x-8`) after 800ms delay.
  - Auto-resizing textarea height matches content.
  - Composer entry animation (fade/zoom).
## Acceptance Criteria
- Key UI components look and feel coherent and Trello-like
- No "unstyled" elements
